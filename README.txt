
markItUp!
------------------------------------

Version: 2.0
Author: Marcin Konicki, Nils Werner
Build Date: 25 January 2009
Requirements: Symphony version 2.0.5 or later.

[SYNOPSIS]

This extension ports jQuery based markItUp! markup editor 
(http://markitup.jaysalvat.com/) to Symphony.

[INSTALLATION]

1. Upload the 'markitup' folder in this archive to your Symphony 'extensions' 
   folder.

2. Enable it on System > Extensions page: select the "markItUp!",
   choose Enable from the "With selected..." menu, then click "Apply" button.

[CONFIGURATION]

You don't need to configure this extension. MarkItUp! will be automatically
attached to all textareas with more than 10 lines of height.
